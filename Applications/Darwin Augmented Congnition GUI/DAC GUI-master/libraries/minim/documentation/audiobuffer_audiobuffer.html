 An AudioBuffer is a buffer of floating point samples corresponding to a single channel of streaming audio. It is readonly, but you can obtain a copy of the samples in the buffer by using the toArray method. In fact, when drawing a waveform, you should use the toArray method rather than iterating over the buffer itself because it is possible that the samples in the buffer will be replaced with new ones between calls to the get method, which results in a waveform that appears to have discontinuities in it.
