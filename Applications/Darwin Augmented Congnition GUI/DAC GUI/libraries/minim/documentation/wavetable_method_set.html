Wavetable

set

Description

Sets the ith entry of the underlying waveform to value. This is equivalent to:

getWaveform()[i] = value;

Signature

void set(int i, float value)

Parameters

i — int: the index of the sample to set
value — float: the new sample value

Returns

None
