 A UGen which simply returns the reciprocal value of it's input. Because this UGen is intended for use with control signals, rather than audio signals, it behaves as a mono UGen, regardless of whether or not it has been configured with more than one channel. This means that the output of Reciprocal will always be the reciprocal of the first (and usually only) channel of the denominator input copied to all output channels, similar to Constant.
